# v0.1.0-beta.1 — Initial Beta Release

CloudLaunch (Go / Wails 版) の初回ベータリリースです。PC ゲームのセーブデータを S3 互換ストレージへ差分同期して、複数の PC で進行状況を持ち運べるようにします。

## できること

- 🎮 **ゲームライブラリ管理**
  - ErogameScape からタイトル / ブランド / カバー画像をインポート
  - ルート単位のセッション管理(攻略 / 周回ごとに分けて記録)
  - プロセス検出による自動プレイ時間計測
- ☁️ **content-addressing 方式のセーブデータ同期**
  - ハッシュベースの差分転送(変更があった blob だけアップロード)
  - 受信時のコンテンツハッシュ検証
  - 別 PC との変更コンフリクト検出
  - Pull 時の untracked ファイル削除を事前確認
  - ゲーム単位のクラウドデータ閲覧 / 削除
- 📝 **Markdown プレイメモ**: リアルタイムプレビュー、ゲーム別整理、クラウド同期
- 🖼️ **スクリーンショット**: ホットキー撮影、ゲーム別ディレクトリへ自動保存
- 💾 **データ管理**: ゲーム情報の JSON エクスポート、DB と関連ファイルのフルバックアップ / 復元
- 🎨 **独自テーマ** `cloudlaunch` (明るめ) を初期搭載
- 📴 **オフラインモード**: クラウド機能を無効化してローカル機能のみで動作

## 動作環境

- **Windows 10 / 11 (x86_64)**
- **WebView2 ランタイム**が必要です。Windows 11 ではプリインストール、Windows 10 でも Windows Update 経由で大半の環境に入っていますが、起動エラーが出る場合は [WebView2 Evergreen Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) をインストールしてください。

## インストール方法

用途に応じてお好きな方をお選びください。

### インストーラ版(推奨)

1. `CloudLaunch-amd64-installer.exe` をダウンロードして実行
2. ウィザードに従ってインストール(スタートメニュー登録 / アンインストーラ付き)

### ポータブル版

1. `CloudLaunch.exe` をダウンロード
2. 任意のフォルダに配置(レジストリへの書き込みはありません)
3. ダブルクリックで起動

## 注意事項

- **ベータ版です**。セーブデータを扱う以上、利用前に必ずローカルでバックアップを取ってください。
- 仕様は変わる可能性があります。データフォーマット(クラウド側 / ローカル DB 双方)の互換性を将来保証するものではありません。
- 認証情報は Windows Credential Manager に保存されます。ファイルや DB には書き出しません。
- macOS / Linux は本リリースに含まれません(プロセス検出 / 認証情報ストア / スクリーンショットが未対応のため対応待ち)。

## 旧 Electron 版からの移行について

Electron 版とはセーブデータの格納形式が異なります。Electron 版から移行する場合は、Electron 版でデータをエクスポートして安全に保管した上で、CloudLaunch (Go 版) で改めてゲーム登録を行ってください。
